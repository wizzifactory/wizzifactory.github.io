Wizzifiers

Some Wizzi Schemas of the default Wizzi Factory plugins have a related wizzifier tool that translates PL code into an ITTF Document of its type.

They are

  • the `js` schema in the `wizzi-js` plugin
  • the `ts` schema in the `wizzi-js` plugin
  • the `html` schema in the `wizzi-web` plugin
  • the `css` schema in the `wizzi-web` plugin
  • the `svg` schema in the `wizzi-web` plugin
  • the `graphql` schema in the `wizzi-web` plugin
  • the `json` schema in the `wizzi-core` plugin
  • the `xml` schema in the `wizzi-core` plugin
