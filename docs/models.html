Wizzi Models

Wizzi models are javascript objects, inspired by Xml document objects, that can load mTrees, applying a type to their nodes.

To be loaded in a wizzi model an mTree node structure must be conformant to the wizzi schema of the model, otherwise an mTreeLoadError is thrown.

Wizzi schemas are described using a meta wizzi model, of schema 'wfschema', implemented by the 'wizzi-core'  plugin. A wizzi schema is a wizzi model data and code description.

Wizzi models are software artifacts generated by artifact generators of the `wizzi-core` plugin. From the definitions of a 'wfschema' model, can be generated:

  • a wizzi model javascript module
  • a model factory javascript module
  • a json documentation file
  • an html documentation file.

Wizzi model artifact generators and links to source code

artifact output file name source code
wfschema/factory {schema-name}-factory.g.js ittf javascript
wfschema/model {schema-name}-model.g.js ittf javascript
wfschema/html_docs {schema-name}-schema.g.html ittf javascript
wfschema/json_docs {schema-name}-schema.g.json ittf javascript

Wizzi models generation

Wizzi models can be generated calling the 'generateModelTypes' method of an instance of the wizzi.WizziFactory class, initizialized with the 'wizzi-core', 'wizzi-js' and 'wizzi-web' plugins.

This is the api of the wizzi.WizziFactory.generateModelTypes method.

  1. ... 
  2.   m generateModelTypes
  3.     string wfschemaIttfDocumentUri

  5.     string outputPackagePath
  6.       # The full path of the package folder where the generated artifacts will be written
  7.     string wfschemaName

  9.     { mTreeBuildUpContext
  10.       # A context object which properties will inserted in the global
  11.       # evaluation context during the mTree loading of the schema
  12.     callback

The wizzi model javascript module

A wizzi model is a javascript module containg a javascript class for each element defined in its 'wfschema' and some standard base and helper classes.

Wizzi model classes

wzSourceLineInfo source map item with source file path, row, col of a node
`schema-name`Exception exception object for this wizzi model
`schema-name`Context context object for the methods wzInitialize and wzVerify
`schema-name`Base base class for all element classes
`element-name` one class for each element of the schema

The wizzi model factory javascript module

The javascript module `schema-name`-factory.g.js exports the method 'createLoadModel', that is an HOF that returns the 'loadModel' function. It must be called passing the 'wizzi' object.
