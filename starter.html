Starter Lab Tutorial

In this breaf tutorial we will build a minimal WizziFactoryPackage.

The package will expose:

  • an ultra-trivialized myhtml LanguageWizziSchema
  • a myhtml/document CodeWriteArtifactGenerator

It will contain a schemas.js module for launching the WizziModelProductions for the package WizziSchemas.

Then we will create a MyHelloSite folder with a myhtml IttfDocument 'Hello world' page that will be transformed to html by a WizziFactoryProductionJob.

myhtml - the first WizziSchema

We want to exploit the power of the Ittf template engine for building html pages, without the need to detail all the tags of the html specification in a WizziSchema.

We would like to write

  1. html 
  2.     < body
  3.         < div Hello world in many colors
  4.         $ var i = 10
  5.         $while i <= 360
  6.             < div Hello world
  7.                 @ style color: hsl(${i}, 100%, 50%);
  8.             $ i++;

We need a very simple schema with three elements only:

  • the html root node
  • the element/< element for html element tags
  • the attribute/@ element for html attributes

A first draft of the WizziSchema could be:

  1. wizzischema myhtml
  2.     e html
  3.         r attribute/s
  4.         r element/s
  5.     e element
  6.         tag <
  7.         r attribute/s
  8.         r element/s
  9.     e attribute
  10.         tag @

It will work but there is a redundancy. We may derive the root html element from di element element.

  1. wizzischema myhtml
  2.     e html : element
  3.     e element
  4.         tag <
  5.         r attribute/s
  6.         r element/s
  7.     e attribute
  8.         tag @
