Source code

                    

                        
This section allow the browsing of the kernel modules and core plugins of the wizzi factory.
Since version v3 (0.3.x pre-release) the source of the wizzi factory code is made of wizzi models, so browsing the source helps to better understand the concepts explained by the documentation.
Ittf notation legend
  1. nodename nodevalue rest of line
  2.   $nodecommand parameters
  3.   mixinCall( param-1, param-2, ... )
        
  • 'nodename' in green is a semantic node name. The meaning of nodename is given by the wizzi schema, if the ittf document is schemed, or is a free meaning, given by the document author.    
  • 'The rest of the line in white is the value of 'nodename'.    
  • '$nodecommand' is a template command. Node commands are documented        here            
  • 'mixinCall' in red is a mixin inclusion command. In the code browser it is clickable and a modal dialog with the content of the mixin is shown.    

                    
Kernel packages

                    

                        
    
                    wizzi        The core package of the wizzi factory.            
                    wizzi-mtree                    Loader for wizzi indented text tree format (ittf) documents.            
                    wizzi-repo                    Repository manager for the wizzi factory.            
                    wizzi-mongodb                    Wizzi mongodb documents storage for the wizzi factory.            
                    wizzi-utils                    Utility library for the wizzi factory.            

                    
Plugin packages

                    

                        
    
                    wizzi-core        A Wizzi Factory plugin with core schemas and artifact generators.            
                    wizzi-meta                    A core wizzi factory plugin with schemas describing wizzi packages and the wizzi studio dashboard.            
                    wizzi-docs                    A wizzi factory plugin for documentation artifact generation.            

                    
The docs package

                    

                        
    
                    v3-docs        The wizzi factory v3 (0.3.x pre-release) documentation.            

                    
The demo package

                    

                        
    
                    wizzi-demo        The most complete demo of the wizzi factory - v3 (pre-release).
